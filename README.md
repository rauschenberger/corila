
# Sparse group lasso

The R package 'corila' (under development) implements the sparse group lasso. This repository has a private pull mirror at the LCSB (https://gitlab.com/uniluxembourg/Personalfolders/armin.rauschenberger/corila) and - once the technical problems are resolved - will also have one at the LIH (https://git.lih.lu/arauschenberger/corila).
